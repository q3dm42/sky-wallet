# Skypro Wallet

SPA-приложение для учета и анализа расходов. Проект собран на React + Vite и сверстан по макетам Skypro Wallet.

## Что реализовано

- Страницы входа и регистрации с шапкой и логотипом Skypro.Wallet.
- Демо-вход: после нажатия на кнопку "Войти" пользователь попадает в приложение.
- Сохранение состояния входа в `localStorage`, после обновления страницы авторизация не сбрасывается.
- Страница "Мои расходы" с таблицей расходов и формой добавления нового расхода.
- Таблица расходов с описанием, категорией, датой, суммой и кнопкой удаления.
- Форма нового расхода с полями описания, даты, суммы и выбором категории.
- Выбор даты через календарь с отображением в формате `дд.мм.гггг`.
- Страница "Анализ расходов" с выбором периода, чипсами категорий и графиком расходов.
- Единые иконки категорий для формы добавления и страницы анализа.
- Адаптивная верстка для основных экранов.

## Страницы

- `/login` - вход.
- `/register` - регистрация.
- `/expenses` - список расходов и добавление расхода.
- `/analysis` - анализ расходов по периоду.

## Технологии

- React
- React Router
- Recharts
- Vite
- ESLint

## Установка и запуск

```bash
npm install
npm run dev
```

После запуска приложение будет доступно по адресу, который покажет Vite в терминале.

## Проверка и сборка

```bash
npm run build
npx eslint src
```

Команда `npm run lint` в текущей конфигурации проверяет весь проект, включая папку `dist`, если она уже создана. Для проверки исходного кода используется `npx eslint src`.
